字符串的介绍

  • 字符串在任何的开发中使用都是非常频繁的
  • OC和Swift中字符串的区别
    • 在OC中字符串类型时NSString,在Swift中字符串类型是String
    • OC中字符串@””,Swift中字符串””

  • 使用 String 的原因

    • String 是一个结构体，性能更高
    • NSString 是一个 OC 对象，性能略差
    • String 支持直接遍历
    • Swift 提供了 String 和 NSString 之间的无缝转换
